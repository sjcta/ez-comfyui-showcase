# 设备管理页面 — 开发规格书 v1.0

## 1. 归属
- 项目: ez-comfyui-showcase
- 分支: 4.0-beta
- 入口: https://imdjj.cn:1313/comfy/
- 相关文件: index.html, style.css, nodes.js, app.py

## 2. 功能清单

### 2.1 入口
- 顶部工具栏新增按钮「🖥️ 设备」，打开设备管理覆盖层

### 2.2 设备列表（覆盖层主视图）
- 展示所有已注册设备卡片
- 每张卡片包含：
  - 设备名称
  - 连接方式标签（本地/SSH/HTTP + 颜色区分）
  - 🟢/🔴 在线/离线状态
  - 主机地址
  - 🔑 SSH 信息按钮（SSH 连接时显示，弹窗展示登录命令）
  - 实例表格（实例名/端口/状态/队列深度/操作按钮）
    - 操作：🔗 打开（新标签页）、■ 停止、▶ 启动
    - 实例状态指示灯：运行中🟢、空闲🟡、已死🔴、离线⚫
  - 底部操作栏：🔍 测连通、🔎 扫端口、✏️ 编辑、🗑 删除
- 顶部工具栏有「+ 添加设备」按钮
- 设备列表每 30 秒自动刷新状态

### 2.3 添加/编辑设备弹窗
- 表单分组：
  - 📋 基础信息：设备名称、主机地址、连接方式（下拉）、标签
  - 🔑 SSH 连接配置（连接方式=SSH时显示）：
    - SSH 用户、SSH 端口、认证方式（密码/密钥）
    - 密码/密钥路径输入框（根据认证方式切换显示）
  - 🌐 访问入口：访问 URL 模板（支持 {host}/{port} 占位符）
  - ⚙️ 实例预设：默认端口列表（逗号分隔）
  - 📡 已登记实例（编辑时显示已有实例列表）
- 保存后关闭弹窗并刷新设备列表
- 表单校验：名称和主机地址为必填

### 2.4 SSH 信息弹窗
- 显示：主机、端口、用户、认证方式
- 自动生成 SSH 登录命令，一键复制

### 2.5 功能按钮
- 🔍 测连通：HTTP/SSH/Systemd 三项检测，结果弹窗显示
- 🔎 扫端口：扫描节点配置的端口范围，结果弹窗显示，可勾选应用
- 🗑 删除：确认后删除

### 2.6 后端 API 依赖
| 方法 | 路径 | 用途 |
|------|------|------|
| GET | /api/nodes | 列出所有设备 |
| GET | /api/nodes/{id} | 单个设备详情 |
| POST | /api/nodes | 添加设备 |
| PUT | /api/nodes/{id} | 编辑设备 |
| DELETE | /api/nodes/{id} | 删除设备 |
| POST | /api/nodes/{id}/test | 连通测试 |
| POST | /api/nodes/{id}/discover | 端口扫描 |
| POST | /api/nodes/{id}/apply-scan | 应用扫描结果 |
| POST | /api/nodes/{id}/instances/{iid}/{start\|stop\|restart} | 实例启停 |
