# Ez ComfyUI Documentation

这个目录集中保存项目 Markdown 文档，根目录只保留代码、配置、脚本和运行入口，避免文档散落。

## 目录

- `project/`：项目说明、开发规范、分支策略等长期有效文档。
- `specs/`：功能规格、UI/UX 规格、修复规格和设计约束。
- `releases/`：版本说明、路线图、阶段记录和 V5.0 移动端整理文档。
- `implementation/`：实施计划、架构迁移、重构管线和任务执行记录。
- `reviews/`：代码审查、修复跟踪和评审补充记录。
- `superpowers/`：执行过程中沉淀的规格和计划文档。

## 常用入口

- [项目 README](project/README.md)
- [开发规范](project/PROJECT_STANDARDS.md)
- [更新日志](releases/CHANGELOG.md)
- [V5.0 移动端整理](releases/v5.0-mobile-agent-overview.md)
