# 06 颜色、风格与材质

## 颜色

- 颜色必须绑定具体对象：肤色、发色、服装主色、服装辅色、背景、道具、光源、阴影和高光。
- 关键颜色尽量写具体色名、近似 HEX、明度、饱和度、冷暖偏色；色值必须放在括号里，例如红色（#D32F2F）。
- 半透明、反光、湿润、丝袜、金属、皮革、玻璃等材质要区分本体色、透出色、反射高光色和阴影色。

## 光线

- 写光源方向、自然光/人造光、硬光/柔光、色温、阴影位置、逆光/侧光/轮廓光/顶光、高光位置。
- 氛围只使用可视化宏观光学现象：体积光、丁达尔光束、光晕扩散、柔焦虚化；不写微尘、漂浮粒子、水雾微粒等不可稳定可视化的微观物。

## 风格

- 风格必须拆成媒介类型、成像来源、视觉语言、后期痕迹和情绪基调。
- 媒介类型可写手机实拍、相机写真、棚拍人像、电影剧照、短视频截图、AI写实、3D渲染、动漫插画、商业广告、生活记录。
- 成像来源写手机感/相机感、景深、锐化、磨皮、美颜、压缩噪点、动态模糊、焦外虚化、镜头畸变、胶片颗粒或数码平滑感。

## 材质

- 写表面纹理、反光特性、透明度、柔软度、织法、褶皱、颗粒、锐度、磨损和边缘细节。
