# 05 暴露内容与 NSFW 细节

## 命名

- 统一使用“暴露内容”描述，不把 suggestive_clothing、visible_exposure、adult_nudity 等内部标签写入正向提示词。
- 只写可见事实，不写吸引力、审美评价或主观性词汇。

## 普通暴露

- 露肩、露背、低领、乳沟、贴身胸部轮廓、短裙、短裤、露腿、丝袜、内衣边缘必须写成衣物、皮肤区域、遮挡边界和可见程度。
- 普通短裤、露出大腿皮肤、长袜、丝袜、日常服装不等于成人裸露。

## 成人裸露与明确性内容

- 成人裸露、乳头、乳晕、性器官、肛门、臀沟、明确性意味接触或液体清晰可见时，直接写可见部位、位置、颜色、遮挡和动作关系。
- 隐私部位细节写可见范围、颜色、红肿、张开、湿润、反光、遮挡物和画面位置。
- 接触动作写接触对象、身体部位或物体、方向、进入/贴压/摩擦等可见关系。
- 液体写颜色、附着位置、湿润反光和接触对象。

## 外观美学描述

- 生殖器外观必须使用“解剖名词 + 外观形态 + 色彩肤色 + 毛发分布 + 皮肤/黏膜质感 + 遮挡关系”的句式，不写“左侧点、右侧点、下面、私处”等含糊词。
- 女性外观可见时逐级区分外阴、阴阜、大阴唇、小阴唇、阴蒂头、阴蒂包皮、尿道口、阴道口、会阴、肛门；只有具体部位本身有独立清晰边界时才写更细名称。
- 女性示例：外阴区域无可见毛发，大阴唇轮廓柔和并贴合，小阴唇仅有细窄边缘可见，整体颜色为浅蜜桃粉，阴道口被阴唇遮挡形成细窄纵向线条。
- 男性外观可见时逐级区分阴茎根部、阴茎体、龟头、冠状缘、包皮、尿道口、阴囊、睾丸轮廓；只有具体部位本身有独立清晰边界时才写更细名称。
- 男性示例：阴茎体部分可见，肤色与腹股沟接近，龟头未外露并被包皮覆盖；阴囊自然下垂，表面有细密皮肤褶皱。
- 肤色和局部颜色使用可视化词：冷白、暖白、象牙白、浅米色、蜜桃粉、浅粉、玫瑰粉、粉棕、浅褐、局部颜色较周围肤色略深、边界柔和。
- 毛发写无可见毛发、稀疏短毛、细软毛发、浓密卷曲毛发、分布集中、边缘修剪整齐；不能用毛发状态推断画外内容。
- 乳房、乳晕和乳头可见时写体积、形态、左右间距、乳晕颜色和边界、乳头大小和突出程度，例如乳房体积中等、轮廓圆润，乳晕浅粉色且边界柔和，乳头小巧突出。

## 年龄边界

- 年龄不明确时，只描述非性化可见暴露和衣物遮挡，不补写成人判断、性化行为、隐私部位细节或液体。
