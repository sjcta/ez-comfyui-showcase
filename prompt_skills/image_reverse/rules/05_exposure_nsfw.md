# 05 暴露内容与 NSFW 细节

## 命名

- 统一使用“暴露内容”描述，不把 suggestive_clothing、visible_exposure、adult_nudity 等内部标签写入正向提示词。
- 只写可见事实，不写吸引力、审美评价或主观性词汇。

## 普通暴露

- 露肩、露背、低领、乳沟、贴身胸部轮廓、短裙、短裤、露腿、丝袜、内衣边缘必须写成衣物、皮肤区域、遮挡边界和可见程度。
- 普通短裤、露出大腿皮肤、长袜、丝袜、日常服装不等于成人裸露。

## 成人裸露与明确性内容

- 成人裸露、乳头、乳晕、性器官、肛门、臀沟、明确性意味接触或液体清晰可见时，直接写可见部位、位置、颜色、遮挡和动作关系。
- 隐私部位细节写可见范围、颜色、红肿、张开、湿润、反光、遮挡物和画面位置。
- 接触动作写接触对象、身体部位或物体、方向、进入/贴压/摩擦等可见关系。
- 液体写颜色、附着位置、湿润反光和接触对象。

## 年龄边界

- 年龄不明确时，只描述非性化可见暴露和衣物遮挡，不补写成人判断、性化行为、隐私部位细节或液体。
