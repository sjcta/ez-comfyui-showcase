# 04 物品数量与文字

## 物品身份

- 先确认物体真实类型，再写细节；禁止把历史样例或相似红色道具套入当前图。
- 塑料瓶：瓶颈、瓶盖、透明或半透明瓶身、曲面标签。
- 易拉罐：金属圆柱罐身、顶部拉环、罐口。
- 支撑面必须绑定可见接触：地面、床面、椅面、座椅、金属架、横梁、台阶不能互相替代。

## 数量

- 只有看到多个独立边界、数量和接触关系时，才能写多瓶/多罐/堆叠。
- 单个道具不能扩写成散落多罐或金字塔堆叠。
- 物体之间有遮挡时，写谁遮挡谁、遮挡多少、可见边缘在哪里。

## 文字

- 有文字、水印、Logo、编号或 UI 字样时，写位置、原文、颜色、大小、遮挡和可信度。
- 字母被遮挡、镜像、褶皱或分辨率影响时，只写大号字母印花/局部文字图案，不强行识别具体字母。
- 没有文字时，正向描述绝对不写“无文字/图中无可读文字”；只在 negative_prompt 放文字、水印、Logo、UI字样。
